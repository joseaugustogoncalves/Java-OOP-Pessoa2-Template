# Projeto Java: Classe Pessoa

## Objetivo
O objetivo deste exercício é implementar uma classe `Pessoa` com métodos simples e aprender os conceitos básicos de Orientação a Objetos em Java.

## Estrutura do Projeto

O projeto já contém:
- Uma classe base `Pessoa` para implementar as suas funcionalidades.
- Testes para garantir que o código está a funcionar corretamente.

### Requisitos:
1. Implemente as funcionalidades na classe `Pessoa`.
2. Certifique-se de que todos os testes na classe `PessoaTest` passem.
3. Na classe `Main`, crie objetos da classe `Pessoa` e demonstre o seu funcionamento.
